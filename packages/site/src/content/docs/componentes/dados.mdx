## Table

Colunas e linhas são fornecidas via propriedades `columns` e `rows` (arrays), por JS.

```html
<ui-table id="tbl" striped></ui-table>
```

```ts
const tbl = document.querySelector("#tbl");
tbl.columns = [
  { key: "nome", label: "Nome" },
  { key: "email", label: "E-mail" },
  { key: "total", label: "Total", align: "right" },
];
tbl.rows = [
  { nome: "Ana", email: "ana@exemplo.com", total: "R$ 120" },
  { nome: "Bruno", email: "bruno@exemplo.com", total: "R$ 80" },
];
```

| Prop        | Tipo           | Padrão  | Notas                                  |
| ----------- | -------------- | ------- | -------------------------------------- |
| `columns`   | array (via JS) | `[]`    | `{ key, label, align? }`               |
| `rows`      | array (via JS) | `[]`    | objetos mapeados por `column.key`      |
| `striped`   | boolean        | `false` |                                        |
| `hoverable` | boolean        | `true`  |                                        |

## Carousel

Os slides são filhos diretos do componente.

<div class="ui-demo">
  <ui-carousel loop style="width: 100%; max-width: 28rem;">
    <div style="padding: 3rem; text-align: center; background: var(--ui-color-surface, #eee);">Slide 1</div>
    <div style="padding: 3rem; text-align: center; background: var(--ui-color-surface, #ddd);">Slide 2</div>
    <div style="padding: 3rem; text-align: center; background: var(--ui-color-surface, #ccc);">Slide 3</div>
  </ui-carousel>
</div>

```html
<ui-carousel loop>
  <div>Slide 1</div>
  <div>Slide 2</div>
  <div>Slide 3</div>
</ui-carousel>
```

| Prop          | Tipo    | Padrão  | Notas                       |
| ------------- | ------- | ------- | --------------------------- |
| `loop`        | boolean | `false` |                             |
| `autoplay`    | number  | `0`     | intervalo em ms; `0` = off  |
| `show-arrows` | boolean | `true`  |                             |
| `show-dots`   | boolean | `true`  |                             |

## Timeline

Combine `ui-timeline` com itens `ui-timeline-item`.

<div class="ui-demo">
  <ui-timeline>
    <ui-timeline-item heading="Pedido criado" time="09:00" variant="accent">
      O pedido foi registrado.
    </ui-timeline-item>
    <ui-timeline-item heading="Pagamento aprovado" time="09:05" variant="success">
      Pagamento confirmado.
    </ui-timeline-item>
    <ui-timeline-item heading="Falha no envio" time="10:30" variant="danger">
      Endereço inválido.
    </ui-timeline-item>
  </ui-timeline>
</div>

```html
<ui-timeline>
  <ui-timeline-item heading="Pedido criado" time="09:00" variant="accent">
    O pedido foi registrado.
  </ui-timeline-item>
  <ui-timeline-item heading="Pagamento aprovado" time="09:05" variant="success">
    Pagamento confirmado.
  </ui-timeline-item>
</ui-timeline>
```

`ui-timeline-item` — `heading`, `time` e `variant` (`accent` `success` `warning` `danger` `muted`).
