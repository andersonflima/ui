## Toast / Toaster

Coloque um `ui-toaster` na página e dispare toasts pelo método `toast()`.

```html
<ui-toaster id="toaster"></ui-toaster>
```

```ts
const toaster = document.querySelector("#toaster");
toaster.toast({
  heading: "Salvo",
  description: "Suas alterações foram salvas.",
  variant: "success",
  duration: 4000,
});
```

Para um toast individual controlado manualmente, use `ui-toast` com `show()` / `close()`.

## Loading Overlay

Cobre o elemento pai (que deve ter `position: relative`).

```html
<div style="position: relative;">
  <!-- conteúdo -->
  <ui-loading-overlay visible label="Carregando…"></ui-loading-overlay>
</div>
```

```ts
const overlay = document.querySelector("ui-loading-overlay");
overlay.visible = true;
```

| Prop           | Valores            | Padrão  |
| -------------- | ------------------ | ------- |
| `visible`      | boolean            | `false` |
| `label`        | string             | —       |
| `spinner-size` | `sm` `md` `lg`     | `md`    |
