## Container

Limita a largura máxima do conteúdo.

```html
<ui-container size="lg">
  <p>Conteúdo centralizado.</p>
</ui-container>
```

| Prop     | Valores                       | Padrão |
| -------- | ----------------------------- | ------ |
| `size`   | `sm` `md` `lg` `xl` `full`    | `lg`   |
| `padded` | boolean                       | `true` |

## Stack

Empilha elementos com espaçamento consistente.

<div class="ui-demo">
  <ui-stack direction="horizontal" gap="sm">
    <ui-badge variant="success">um</ui-badge>
    <ui-badge variant="info">dois</ui-badge>
    <ui-badge variant="warning">três</ui-badge>
  </ui-stack>
</div>

```html
<ui-stack direction="horizontal" gap="sm">
  <ui-badge variant="success">um</ui-badge>
  <ui-badge variant="info">dois</ui-badge>
</ui-stack>
```

| Prop        | Valores                                | Padrão    |
| ----------- | -------------------------------------- | --------- |
| `direction` | `vertical` `horizontal`                | `vertical`|
| `gap`       | `xs` `sm` `md` `lg` `xl`               | `md`      |
| `align`     | `start` `center` `end` `stretch`       | `stretch` |
| `justify`   | `start` `center` `end` `between`       | `start`   |
| `wrap`      | boolean                                | `false`   |

## Grid

```html
<ui-grid columns="3" gap="md">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</ui-grid>
```

| Prop             | Tipo   | Padrão | Notas                          |
| ---------------- | ------ | ------ | ------------------------------ |
| `columns`        | number | `12`   |                                |
| `gap`            | enum   | `md`   | `xs` `sm` `md` `lg` `xl`       |
| `min-item-width` | string | —      | habilita auto-fit responsivo   |

## Navbar

Use os slots `brand`, padrão (centro) e `actions`.

```html
<ui-navbar>
  <span slot="brand">Minha Marca</span>
  <a href="#">Produto</a>
  <a href="#">Preços</a>
  <div slot="actions">
    <ui-button variant="solid">Entrar</ui-button>
  </div>
</ui-navbar>
```
