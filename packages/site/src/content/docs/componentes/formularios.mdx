### Prefix e suffix

Use os slots `prefix` e `suffix` para ícones ou unidades.

<div class="ui-demo">
  <ui-input placeholder="Buscar">
    <span slot="prefix">🔎</span>
  </ui-input>
  <ui-input placeholder="Valor">
    <span slot="prefix">R$</span>
    <span slot="suffix">,00</span>
  </ui-input>
</div>

```html
<ui-input placeholder="Buscar">
  <span slot="prefix">🔎</span>
</ui-input>
```

## Radio Group

As opções são fornecidas via propriedade `options` (array), por JS.

```html
<ui-radio-group id="plano" name="plano"></ui-radio-group>
```

```ts
const el = document.querySelector("#plano");
el.options = [
  { label: "Mensal", value: "monthly" },
  { label: "Anual", value: "yearly" },
];
el.value = "monthly";
```

## Select

As opções são fornecidas via propriedade `options` (array), por JS.

```html
<ui-select id="pais" placeholder="Selecione um país"></ui-select>
```

```ts
const el = document.querySelector("#pais");
el.options = [
  { label: "Brasil", value: "br" },
  { label: "Portugal", value: "pt" },
  { label: "Estados Unidos", value: "us" },
];
el.value = "br";
```
