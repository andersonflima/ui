```html

```

### Propriedades

| Prop      | Valores                                                             | Padrão     |
| --------- | ------------------------------------------------------------------ | ---------- |
| `variant` | `solid` `default` `subtle` `outline` `ghost` `destructive` `link`  | `default`  |
| `size`    | `sm` `md` `lg` `icon`                                              | `md`       |
| `loading` | boolean                                                            | `false`    |
| `full`    | boolean                                                            | `false`    |
| `disabled`| boolean                                                            | `false`    |
